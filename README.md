# 🌦️ Weather App

## 📌 Overview
A modern Android Weather App built using Kotlin that provides real-time weather updates based on user location or searched cities. The app displays temperature, weather conditions, and other useful details in a clean and user-friendly interface.

---

## 🚀 Features
- 🌍 Get weather by current location
- 🔍 Search weather by city name
- 🌡️ Real-time temperature updates
- ☁️ Weather conditions (Clouds, Rain, Clear, etc.)
- 📱 Clean and responsive UI
- ⚡ Fast API-based data fetching

---

## 🛠️ Tech Stack
- **Kotlin**
- **MVVM Architecture**
- **Retrofit (API calls)**
- **Weather API (e.g., OpenWeatherMap)**
- **Jetpack Components**
- **Coroutines / Flow**
